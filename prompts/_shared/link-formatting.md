# Link formatting (workspace and Graph)

Use markdown **`[visible file name](url)`** only:

- **Link text** = basename / Graph item `name` only.
- **URL** appears **only** inside `(...)` as the href.
- Do **not** print the same URL again as raw text on the following line.
- Never append `&action=edit` (or similar) to the visible file name.

**Graph (OneDrive / SharePoint):** URL from `open_url` or `webUrl`.

**Local workspace:** URL = `file_uri` from `search_workspace_text` or `workspace_file_link`.

When listing **three or more** files, prefer a markdown table with columns **File** | **Location** | **Note** and a header separator row.
