# Registry and servers

- `registry/servers.yaml` defines MCP servers.
- `registry/agents.yaml` defines agents and their allowed servers.
- `src/crisai/runtime.py` builds server connections.
- `src/crisai/servers/workspace_server.py` is the local workspace MCP server.
- `src/crisai/servers/diagram_server.py` is the local diagram MCP server.
