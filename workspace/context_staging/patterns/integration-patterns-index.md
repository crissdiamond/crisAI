## Design overview
- Catalogue name: Integration patterns.
- The catalogue groups patterns into Consumer, Producer, and Ingestion.
- Consumer patterns:
  - Pattern 0 - Direct
  - Pattern 1 - Enterprise Channel to System : Synchronous
  - Pattern 2 - Enterprise Channel to System : Asynchronous
  - Pattern 3 - Enterprise API to System : Synchronous
  - Pattern 4 - Enterprise API to System : Asynchronous
- Producer patterns:
  - Pattern 1 - System to Enterprise API : onDemand Synchronous
  - Pattern 2 - System to Enterprise Channel : Batch Synchronous
  - Pattern 3 - System to Enterprise Channel : onChange Synchronous
- Ingestion patterns:
  - Pattern 1 - Database to Ingestion API - Draft
  - Pattern 2 - API to Ingestion API - Draft
  - Pattern 3 - DMS to Ingestion API - Draft
