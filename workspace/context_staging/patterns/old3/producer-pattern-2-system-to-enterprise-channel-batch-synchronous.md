## Design overview
- Name: System to Enterprise Channel : Batch Synchronous.
- Description: fetch data from a system on a scheduled basis and send over the Enterprise Channel.
- Classification: producer pattern.

## When to use
- Use when scheduled fetching from a system and delivery over the Enterprise Channel is required.

## Implementation
- Input: data from a system.
- Trigger: scheduled basis.
- Action: fetch data and send over the Enterprise Channel.
- Delivery mode: batch synchronous.
