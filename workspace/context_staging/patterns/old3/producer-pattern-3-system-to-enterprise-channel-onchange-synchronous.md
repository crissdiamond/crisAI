## Design overview
- Name: System to Enterprise Channel : onChange Synchronous.
- Description: system events are received and sent over the Enterprise Channel.
- Classification: producer pattern.

## When to use
- Use when system events should be relayed over the Enterprise Channel as they change.

## Implementation
- Input: system events.
- Action: receive events and send over the Enterprise Channel.
- Delivery mode: onChange synchronous.
